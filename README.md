# Just run this
## First
- ```curl https://get.docker.com/ | sh```
- ```curl https://get.k3s.io/ | sh```

## Then:
- ```kubectl apply -k ./```

# Alternative

You can just download the get docker and the get k3s scripts and then run them manually
## So run...
- ```chmod u+x *.sh```
- ```./get-docker.sh```
- ```./get-k3s.sh```
